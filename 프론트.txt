자바 스크립트 변수 선언 var, let, const
var : 최초 선언된 변수가 전역변수로써 기능을 수행
let, const 는 { } 블록 내의 범위를 가진다.
var, let 은 값을 변경할 수 있지만 const 는 불가능하다.
